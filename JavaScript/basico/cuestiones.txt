1. ¿Cómo puedo hacer para que se muestre el texto “Tu navegador no permite la ejecución de scripts” 
cuando el navegador no soporte la ejecución de scripts?

Agregar la etiqueta <noscript> El navegador no soporta la ejecución de scripts. </noscript> tras
la etiqueta de inicio del body de fichero HTML.
